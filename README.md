## Setup

```sh
$ make install
```

## Run tests

```sh
$ make test
```

## Usage
```
gendiff -h

  Usage: gendiff [options] <firstConfig> <secondConfig>

  Compares two configuration files and shows a difference.

  Options:

    -h, --help           output usage information
    -V, --version        output the version number
    -f, --format [type]  Output format
```
